## Phishing Awareness Training Analysis  
In-Person vs. Online Training Effectiveness

This report analyzes whether in-person phishing awareness training is more effective than online training for GitHub programmers. It compares short-term performance and long-term retention of phishing detection skills.

## Food.com — Restaurant Search and Reservation Web Application

Food.com is a full-stack web application that allows users to search for restaurants by name, location, category, price, and rating, and to make reservations. The application integrates Flask with Neo4j and PostgreSQL to display restaurant details, reviews, and reservation functionality.

## Citi Bike Sentiment Analysis

This notebook applies natural language processing techniques to analyze user sentiment toward Citi Bike. The analysis identifies key themes and emotional trends, translating rider feedback into actionable insights for service and user experience improvement.

## MTA 96th Street Station Simulation and Sensitivity Analysis

This project builds a discrete-event simulation of passenger flow at the 96th Street subway station to evaluate congestion and wait times. Sensitivity analysis examines how changes in demand and service parameters impact overall system performance.
