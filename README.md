Phishing awareness training analysis: the effectiveness of in-person vs. online trainings
This report analyzes whether in-person phishing awareness training is more effective than online training for GitHub programmers, comparing short-term performance and long-term retention of phishing detection skills. 
  
Food.com: a restaurant search and reservation web application
This project implements Food.com, a full-stack web app that lets users search for restaurants by name, location, category, price, and rating, and make reservations. It integrates Flask with Neo4j and PostgreSQL to display restaurant details, reviews, and reservation functionality.
